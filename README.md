# 旺旺 (WangWang) - AI Workflow Canvas

> ⚠️ **Development Status**: 本项目处于持续迭代中（WIP），功能与接口可能会调整。

## 简介 (Introduction)

旺旺是一个基于 React Flow 的可视化 AI 工作流编排平台。你可以在无限画布中通过拖拽节点（文本、图片、图片生成、视频生成等）搭建创作流程，完成从提示词到内容产出的闭环。

它适用于多种创作场景，例如：
- 动漫短片/镜头段落创作：用图片/参考图定义主体与风格，再生成短视频片段
- 广告设计与电商素材：快速生成产品海报、KV、分镜与动效片段
- 内容生产工作流：把“灵感 → 文案 → 画面 → 视频”拆成可复用的节点链路

## 核心功能 (Core Features)

- **可视化画布**: 基于 React Flow 的无限画布，支持节点拖拽、缩放、连线。
- **多类型节点**:
  - 📝 **文本节点**: 文本输入，支持与其它节点连线作为上游提示词来源。
  - 🖼️ **图片节点**: 支持图片上传、旋转、缩放、预览。
  - ✨ **图片生成节点**: 文生图/图生图（单图/多图参考），生成结果可连线输出到图片节点展示。
  - 🎬 **视频生成节点**: 文生视频、首帧图生、首尾帧图生、参考图图生；支持音频开关、分辨率、时长、比例等参数。
- **交互体验**:
  - 双击空白处/连线拖拽创建新组件。
  - 节点自定义工具栏（旋转、缩放等）。
  - 深色模式 UI 设计。
- **模型与接口配置**:
  - 在设置页按“文本/图片/视频”分类配置服务商参数（Base URL、API Key、模型列表）。
  - 模型列表支持启用/停用，并可通过“双击模型设为默认”。
  - 支持配置有效性测试，便于快速验证连通性。
- **火山引擎（方舟）对接**:
  - 图片：Seedream（OpenAI 兼容 `images/generations`）
  - 视频：Seedance（异步任务 `contents/generations/tasks` + 轮询查询）

## 技术栈 (Tech Stack)

- **Frontend**: React, TypeScript, Vite, Tailwind CSS, React Flow (@xyflow/react)
- **Backend**: Express.js (轻量级后端，用于代理 API 请求和项目存储)
- **Icons**: Lucide React


2.  **启动开发服务器**:

## 使用指引 (How to Use)

### 1) 配置服务商
- 进入设置页：分别在“图片模型 / 视频模型”中添加“火山引擎”配置
- 填写 Base URL（默认已提供 `https://ark.cn-beijing.volces.com/api/v3`）与 API Key
- 在“模型列表”中启用可用模型，并双击设为默认模型
- 点击“测试”验证配置可用

### 2) 图片创作工作流
- 在画布中添加“图片生成”节点
- 连接文本节点作为提示词（可选）
- 连接 1~N 张图片节点作为参考图（可选）
- 生成后把输出连到图片节点查看结果

### 3) 视频创作工作流
视频生成是异步任务：创建任务后会自动轮询状态，完成后展示视频播放地址。
- 添加“视频生成”节点
- 文生视频：不连接图片节点，仅输入文本提示词
- 图生视频（首帧）：连接 1 张图片节点
- 图生视频（首尾帧）：连接 2 张图片节点（按画布位置从上到下排序作为首/尾帧）
- 图生视频（参考图）：连接 3 张及以上图片节点作为参考图
